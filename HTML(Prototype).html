Historical Perspective on Computing: Discuss the evolution of computing

Looking back at time, computing goes way back onto the old times. For example, the abacus is a simple babylonian ivention that uses beads as a way of calculating. Moving to the 19th century where in Joseph Marie Jacquard made the "Jacquard Machine" which inspired the following development of computers, it is an automated loom that uses punch cards to automatically weaves silk cloth.


In 1820's, Charles Babbage started his first invention called the "Difference Engine" which was powered by steam, this project however failed because the lack of funding from the government. Years later, he made his own second invention called "Analytical Engine" which has been discussed as the first programmable computer using punch cards. At this time, Charles met Ada Lovelace(Deemed to be the First Programmer), Lovelace became interested to the project of Charles Babbage which she helped by creating an algorithm to help calculate Bernoullis Numbers.


In 1942, John Atanasoff and his student Clifford Berry made the first digital computer.


1942, The Colossus Mark I was invented in collaboration with Alan Turing (Father of Modern Computer Science and AI| also deciphered the enigma code using Bombe) which deciphers the german encrypted messages during world war II. It is considered to be the first fully programable digital computer.


In 1945, the ENIAC (Electronic Number Integrator and Computer) was invented by John Mauchly and J. Presert Ecker. The ENIAC was declared as the first electronic digital computer. Despite being programmable, its process to calculate took long as wires needed to be rewired for a new set of calculation. Around this time, Jon Von Neumann an American Mathematician, proposed the "Jon Neumann Architecture" which founded how computers of today was built. Introduction to IT Basics: Provide an overview of fundamental IT concepts,

(1) hardware,

- Hardware is a physical components of a computer that is used to either entering data using an input device(e.g.Keyboard, Mouse, Microphone,) or presenting the product of the data by an output device (e.g. Speaker, Monitor, Printer). The input data that was entered by the input device will then be stored and processed by the processing components of the computer (e.g. RAM, cpu, gpu, ssd/harddisk, which would then present or display the output data/product using a output device.


>(2) software,

- Software is a application that is a set of programmed code that acts as instructions for the computer to follow. There are two types of software applications 1. Application Software and 2. Systems Software. [APPLICATION SOFTWARE] is used to make everyday life faster and efficient compared to the traditional ways of doing task which would take a longer time compared to using an application. [SYSTEMS SOFTWARE] focuses on the operating system of the computer (e.g. windows, macOS, linux), drivers, utilities. The primary purpose of operating system is to help managing computer data and help softwares communicate with hardwares.


(3) networking,

- Networking is the connection between computers for the purpose of sending, sharing, or storing information/resources from one computer(e.g. personal desktop) to another(e.g. mobile phone, internet, server,) which could be done through a physically using a cable or wirelessly by signals.
